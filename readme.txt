This is a repositoy containing the solutions for the Technical test for Quilk

To run this please install all needed node modules by running 
** npm install 


Go to the src directory and RUN the angular project by typing 
** ng serve 

Open your favourite browser and go to localhost:4200 to see the solutions for the questions


To Run your angular tests 
Please type in the following in a new cmd
** ng test
